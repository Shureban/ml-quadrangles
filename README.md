# Поиск четырехугольников в наборе координат #

- Задание: посчитать количество прямоугольников
- Входящие данные: массив точек с координатами x и y
- Необходимая версия Golang: 1.13.*

Задача решена с использованием класификатора KNN(K ближайших соседей). 
Обучение модели проходило на координатах, представленных в файле coordinates.json.
Набор решающих признаков представлен в виде списка булевых значений (отвечающих за свойства четырехугольников), для простоты исполнения.
При необходимости можно заменить более сложные, вычисляемые параметры.

Модель можно дообучить для определения остальных видов четырехугольников: трапеция, равнобокая трапеция...

Для того, чтобы указать свой набор координат, измените файл coordinates.json, в соответствующем формате.
